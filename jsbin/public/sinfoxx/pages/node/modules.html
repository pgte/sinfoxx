Modules

Em Node.js, um módulo corresponde a um ficheiro, geralmente contendo código JavaScript.

Vamos criar um módulo:

modules/module_a.js:


module.exports = 'A';

Vamos agora requerer o módulo:


→ node
> var A = require('./module_a');
undefined
> a
'A'

Vamos criar outro módulo:

modules/make_urgent.js:


module.exports =
function makeUrgent(s) {
  return s.toUpperCase() + '!!!';
};

E usá-lo:


→ node
> var makeUrgent = require('./make_urgent');
undefined
> makeUrgent('pass me the salt');
'PASS ME THE SALT!!!'

Um módulo também pode exportar um objecto:

modules/big_module.js:


module.exports = {
  makeUrgent: require('./make_urgent'),
  a: require('./module_a'),
  pi: Math.PI
};

E podemos usá-lo agora:


> var bigModule = require('./big_module');
undefined
> bigModule
{ makeUrgent: [Function: makeUrgent],
  a: 'A',
  pi: 3.141592653589793 }
> bigModule.makeUrgent('hello');
'HELLO!!!'
