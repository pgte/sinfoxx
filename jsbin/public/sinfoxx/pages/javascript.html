JavaScript

JavaScript, JS ou ECMAScript:

  • Uma linguagem interpretada
  • Foi criada em 1994 por Brendan Eich
  • Para manipular documentos em web browsers

É uma linguagem:

  • Interpretada
  • Dinâmica
  • Weakly-typed ou loose-typed
  • First-class functions

Fast-forward para 2013, JavaScript é usado fora do contexto de páginas web em:

  • Aplicações Desktop (Windows8, Gnome, etc.)
  • Servidores (Netscape Enterprese Server, Node.js e outros)
  • Bases de Dados (CouchDB, MongoDB)

Standardização:

  • ECMAScript 1 (1997)
  • ECMAScript 2 (1998)
  • ECMAScript 3 (1999) - Adicionadas expressões regulares, melhoramentos às strings, novas estruturas de controlo, try / catch e outras melhorias.
  • ECMAScript 4 - abandonada
  • ECMAScript 5 (2009) - strict mode, getters e setters, JSON, reflection
  • Harmony - WIP
